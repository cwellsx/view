React Forum

This is an experiment -- write a discussion forum, using React.

It is a "minimum viable" implementation, a proof of concept -- one could add more features, but the most essential are in place.

There are three types of data -- Discussions, Tags, and Users.

Data Pages
Discussions
  • List of All discussions -- this can be sorted by "newest" (when it was created), or "active" (when its most recent message was posted); and there's pagination (e.g. 50 per page).
  • Each discussion, for example here -- there's a text area at the end where you can post another answer.
  • Start a New discussion -- if you click the "Ask Question" button on that page, you'll see that the user's data entry is validated; and there's a tag editor, which displays existing tags as hints.
Tags
  • List of All tags
  • Discussions for each tag
  • Description of each tag
  • And a page to Edit the description
Users
  • List of All users.
  • For each specific users, three tabs:
    • Profile
    • Edit settings
    • Activity
